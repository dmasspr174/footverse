# 👟 Footverse — Product Catalog Web App

A modern product catalog web application built with React.js that dynamically displays product data from external APIs with a fast and responsive user experience.

---

## 🚀 Live Demo
https://footverse.vercel.app/

---

## 👨‍💻 My Role
Frontend Developer

---

## 🧠 Overview
Footverse is a Single Page Application (SPA) designed to simulate a modern e-commerce product catalog. The application consumes external API data and transforms it into a structured product interface with smooth navigation and responsive UI.

---

## ⚙️ Key Features
- 🛍️ Dynamic product rendering from external API  
- 🔎 Category-based filtering system  
- ⚡ Fast navigation using SPA (no page reload)  
- 📱 Fully responsive design (mobile, tablet, desktop)  
- 🔄 Real-time data fetching  

---

## 🛠️ Tech Stack
- React.js (Vite)  
- Tailwind CSS  
- REST API (Pixabay)  
- JavaScript (ES6+)  
- Vercel (Deployment)  

---

## 🧠 Technical Highlights
- Implemented React Hooks (`useState`, `useEffect`) for state and lifecycle management  
- Data fetching and transformation from API into structured product objects  
- Modular component architecture for scalability and maintainability  
- Client-Side Rendering (CSR) for faster user interaction

## 🔄 Application Flow
1. User opens the application  
2. App sends request to external API (Pixabay)  
3. Data is fetched and stored in state  
4. Data is mapped into product structure  
5. UI updates dynamically  

---

## 📈 Impact
- Improves browsing experience with fast and smooth navigation  
- Reduces page load time using SPA approach  
- Demonstrates efficient API integration in frontend applications  

---

## 🎯 Project Value
This project demonstrates my ability to build a frontend application that integrates external APIs, manages dynamic data, and delivers a responsive and scalable UI.

---

## ⚠️ Limitations
- No backend or database integration  
- No authentication system  
- Fully dependent on external API
